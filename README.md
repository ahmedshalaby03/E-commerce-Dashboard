# Admin Dashboard UI 📊

تصميم لوحة تحكم (Dashboard) احترافية ومتجاوبة، بتعرض إحصائيات المستخدمين، المنتجات، والأرباح بشكل منظم.

## ✨ المميزات:
* **Dashboard Stats**: عرض سريع لأعداد المستخدمين، البوستات، والمنتجات.
* **Product Table**: جدول تفصيلي للمنتجات مع الأسعار والكميات.
* **Sidebar Navigation**: قائمة جانبية بتصميم عصري للتنقل بين الأقسام (Clients, Charts, Settings).

## 🛠 التكنولوجيات:
* **HTML5 & CSS3**
* **Responsive Design**
